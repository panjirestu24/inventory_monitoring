❓ TROUBLESHOOTING

❌ Database gagal connect
   → Cek MySQL Laragon sudah jalan (lampu hijau)
   → Cek database db_inventory sudah diimport
   → Cek config/database.php (username/password)

❌ Data tidak muncul
   → Tekan F12 di browser → tab "Console" → cek error
   → Tab "Network" → cek request API (harus status 200)

❌ Halaman blank/putih
   → Cek C:\laragon\logs\apache_error.log

❌ Chart tidak muncul
   → Cek koneksi internet (Chart.js pakai CDN)
